# Smart-Disposer-AI (Next-Gen Kitchen Intelligence)

本项目旨在基于 **ESP32-S3** 硬件平台，开发超越硅谷竞争对手 **Mill** 的下一代智能厨余处理器方案。

## 🚀 研发路线图 (Roadmap)

### 第一阶段：感知与交互 (P0: 核心卖点)

| 周期 | 功能名称 | 难度 | 技术实现 (S3 特性) | 商业价值 (对比 Mill) |
| :--- | :--- | :--- | :--- | :--- |
| **W1** | **WiFi CSI 空间感知** | ⭐ | 利用 WiFi 信号扰动检测人体靠近 | **免传感器唤醒**：人走近屏亮，增加科技感 |
| **W2** | **全离线语音控制** | ⭐⭐ | ESP-Skainet 语音唤醒词识别 | **卫生交互**：双手油腻时语音控制，Mill 不支持 |
| **W3** | **FFT 异物碰撞检测** | ⭐⭐⭐ | 实时频谱分析 (FFT) 捕捉金属撞击声 | **安全保护**：毫秒级停机保护刀头，Mill 靠过载 |

### 第二阶段：智能优化 (P1: 能效与体验)

| 周期 | 功能名称 | 难度 | 技术实现 (S3 特性) | 商业价值 (对比 Mill) |
| :--- | :--- | :--- | :--- | :--- |
| **W4** | **研磨指纹识别** | ⭐⭐⭐ | 声音频率回归算法判断研磨细腻度 | **极致节能**：磨好即停，不浪费一度电 |
| **W5** | **手势感应控制** | ⭐⭐⭐ | 利用摄像头进行手势轨迹识别 | **多模态交互**：嘈杂环境下通过挥手操作 |
| **W6** | **视觉容量监测** | ⭐⭐⭐⭐ | 深度图/摄像头评估桶内剩余容积 | **防溢出提示**：实时告知用户还能装多少 |

### 第三阶段：大模型与生态 (P2: 护城河)

| 周期 | 功能名称 | 难度 | 技术实现 (S3 特性) | 商业价值 (对比 Mill) |
| :--- | :--- | :--- | :--- | :--- |
| **W7** | **视觉垃圾分类** | ⭐⭐⭐⭐ | ESP-DL 运行轻量级识图模型 | **智能策略**：区分骨头/纤维，自动调节转速 |
| **W8** | **AI 故障自诊断** | ⭐⭐⭐ | 异常声学检测 (Anomaly Detection) | **售后降本**：自检故障原因，减少盲目上门 |
| **W9** | **云端 LLM 顾问** | ⭐⭐ | ESP-HTTPS 将图片上传至大模型 | **厨余百科**：回答分类知识，增加用户粘性 |
| **W10** | **综合闭环系统** | ⭐⭐⭐⭐ | 全系统集成演示 | **全链路打击**：展示产品的系统化能力 |

## 🛠️ 硬件环境 (Hardware)

* **ESP32-S3-BOX**: 核心交互端，负责语音、屏幕显示及音频采集。
* **ESP32-S3-CAM**: 视觉采集端，负责物体分类与扫码。

📂 目录结构说明
/S3-BOX-Audio-Core: 语音唤醒、FFT 音频监测。

/S3-CAM-Vision-Core: 图像识别、垃圾分类模型。

/S3-Sensor-Fusion: WiFi CSI、电流/震动多传感器融合。

/docs: 技术方案书、演示视频及对比测试数据。

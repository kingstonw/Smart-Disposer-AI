# Smart-Disposer-AI (Next-Gen Kitchen Intelligence)

[![Hardware: ESP32-S3](https://img.shields.io/badge/Hardware-ESP32--S3-orange)](https://www.espressif.com/en/products/socs/esp32-s3)
[![Platform: ESP-IDF](https://img.shields.io/badge/Platform-ESP--IDF-blue)](https://docs.espressif.com/projects/esp-idf/en/latest/esp32s3/index.html)
[![Benchmark: Mill-Focused](https://img.shields.io/badge/Benchmark-Mill-green)](https://www.mill.com)

本项目致力于基于 **ESP32-S3** 硬件平台，通过边缘 AI (Edge AI) 算法赋能，打造在**感知深度、交互实时性、运行安全性**上全面超越硅谷标杆 **Mill** 的下一代智能厨余处理方案。

---

## 🚀 研发路线图 (Roadmap)

我们将研发分为四个阶段，由浅入深榨取硬件潜力，最终实现全场景闭环。

### 第一阶段：交互与基础安全 (P0: 核心功能)
| 周期 | 功能名称 | 技术实现 (S3 特性) | 商业价值 (对比 Mill) |
| :--- | :--- | :--- | :--- |
| **W1** | **WiFi CSI 空间感知** | WiFi CSI 信号扰动算法 | **无感唤醒**：人走近屏亮/开盖，极致交互体验 |
| **W2** | **全离线语音指令集** | ESP-Skainet 语音唤醒/指令 | **卫生交互**：双手沾满油腻时 0 延迟声控 |
| **W3** | **FFT 异物碰撞检测** | 实时音频频谱分析 (FFT) | **硬件保护**：毫秒级识别金属误入并停机，保护刀头 |
| **W4** | **Rust重构部分接口** | Rust(Zero-copy Parser)|**工业级稳定性**：按照美国政府标准，消除内存溢出风险，确保 24/7 零故障运行，降低售后成本。|

### 第二阶段：精准控制与运营优化 (P1: 深度性能榨取)
| 周期 | 功能名称 | 技术实现 (S3 特性) | 商业价值 (对比 Mill) |
| :--- | :--- | :--- | :--- |
| **W5** | **研磨指纹识别** | 声音频率特征回归算法 | **精准停机**：识别研磨完成声纹，磨好即停，极度节能 |
| **W6** | **负载预测与自愈** | ADC 电流采样 + 时序分析 | **自动处理**：模仿 OpenClaw 逻辑，实现堵转自动反转自愈 |
| **W7** | **极低功耗视觉哨兵** | Deep-sleep + RTC 快速唤醒 | **安全监控**：休眠下监测非法投物，极致待机表现 |
| **W8** | **智能数据瘦身 (MQTT)** | 边缘变化检测与冗余过滤 | **运维降本**：减少 80% 冗余消息，大幅降低云端存储成本 |

### 第三阶段：专项降维打击 (P1: 差异化卖点)
针对 Mill 的现有短板，利用多传感器融合技术实现功能反超：

| 周期 | 功能名称 | 技术实现 (S3 特性) | 销售卖点 (Sales Point) |
| :--- | :--- | :--- | :--- |
| **W9** | **智能防溢出** | 视觉 + 超声波融合算法 | 实时识别余容，预测是否能容纳当前投入物，避免卡死 |
| **W10** | **生物入侵报警** | CSI 波动识别 + 移动侦测 | 监测蟑螂/果蝇侵入并自动密封，直击厨房卫生痛点 |
| **W11** | **滤芯精准监测** | AI 电子鼻 (Gas Fingerprint) | 监测排气质量，按需更换，降低用户后期耗材成本 |

### 第四阶段：生态、视觉与大数据 (P2: 技术护城河)
| 周期 | 功能名称 | 技术实现 (S3 特性) | 商业价值 |
| :--- | :--- | :--- | :--- |

| **W13** | **AI 故障自诊断** | 异常声学检测 (Anomaly) | **主动售后**：“听诊”机器状态，提前预警轴承磨损 |

| **W15** | **全系统集成测试** | 软硬件综合联调 | 确保多模态算法协同工作，系统稳定性验证 |

## 🛠️ 硬件支撑 (Hardware Stack)
* **ESP32-S3-BOX**: 核心交互中枢（语音、UI、音频 AI）。
* **ESP32-S3-CAM**: 视觉感知中枢（垃圾识图、哨兵模式、容量监测）。
* **Sensor Fusion**: 集成电流互感器、超声波测距、BME688 气体传感器。

## 📂 目录结构 (Structure)
* `/S3-BOX-Audio-Core`: 语音唤醒、FFT 异物监测、研磨声纹特征库。

## 📈 商业价值汇报 (Management Summary)
1. **运营优势**：边缘过滤算法显著降低云端服务器带宽与存储开销（降本 80%）。
2. **可靠性**：类似 OpenClaw 的本地自愈能力与异物听诊功能，极大延长设备寿命。
3. **差异化竞争**：通过生物监测、滤芯气体分析等 Mill 缺失的功能，建立核心技术壁垒。
